What is Flexbox?

Flexbox (Flexible Box) is a CSS layout model designed to create responsive and dynamic layouts. It allows elements to grow, shrink, and align themselves within a container.

🔑 Key Concept: Flex Container vs Flex Items

  • Flex Container: The parent element with display: flex
  • Flex Items: The direct children of the flex container

📋 Quick Reference Summary

Container Properties

  • display: flex - Enable flexbox
  • flex-direction - row, row-reverse, column, column-reverse
  • flex-wrap - nowrap, wrap, wrap-reverse
  • justify-content - Main axis alignment
  • align-items - Cross axis alignment

Item Properties

  • flex-grow - How much to grow
  • flex-shrink - How much to shrink
  • flex-basis - Initial size
  • flex - Shorthand for all three
  • align-self - Override container alignment
