📋 Flexbox Quick Reference Guide

Your complete cheat sheet for Flexbox properties

🔧 Container Properties

display: flex

Enables flexbox for the container. All direct children become flex items.

/* Makes the element a flex container */
.container {
  display: flex;
}

flex-direction

Defines the direction flex items are placed in the container.

row row-reverse column column-reverse
/* Items in a horizontal row */
.container {
  flex-direction: row;
}

flex-wrap

Controls whether flex items should wrap or not.

nowrap wrap wrap-reverse
/* Items wrap onto multiple lines */
.container {
  flex-wrap: wrap;
}

justify-content

Aligns flex items along the main axis (horizontal in row direction).

flex-start flex-end center space-between space-around space-evenly
/* Items centered horizontally */
.container {
  justify-content: center;
}

align-items

Aligns flex items along the cross axis (vertical in row direction).

flex-start flex-end center baseline stretch
/* Items centered vertically */
.container {
  align-items: center;
}

align-content

Aligns lines within a flex container when there's extra space in the cross-axis.

flex-start flex-end center space-between space-around stretch

gap

Sets the space between flex items.

/* 20px gap between items */
.container {
  gap: 20px;
}

📦 Flex Item Properties

flex-grow

Defines the ability for a flex item to grow if necessary. Default is 0.

/* Item will grow to fill available space */
.item {
  flex-grow: 1;
}

flex-shrink

Defines the ability for a flex item to shrink if necessary. Default is 1.

/* Item won't shrink */
.item {
  flex-shrink: 0;
}

flex-basis

Defines the default size of an element before remaining space is distributed.

/* Initial size of 200px */
.item {
  flex-basis: 200px;
}

flex

Shorthand for flex-grow, flex-shrink, and flex-basis combined.

/* flex: grow shrink basis */
.item {
  flex: 1 0 200px;
}

/* Common shorthand */
.item { flex: 1; } /* flex: 1 1 0% */
.item { flex: auto; } /* flex: 1 1 auto */
.item { flex: none; } /* flex: 0 0 auto */

align-self

Allows the default alignment (or the one specified by align-items) to be overridden.

auto flex-start flex-end center baseline stretch
/* This item will be aligned to the end */
.item {
  align-self: flex-end;
}

order

Controls the order in which flex items appear in the flex container.

/* Item will appear first regardless of HTML order */
.item {
  order: -1;
}

⚡ Quick Cheat Sheet

🎯 Centering

  • display: flex
  • justify-content: center
  • align-items: center

📐 Space Between

  • display: flex
  • justify-content: space-between

📱 Responsive Grid

  • display: flex
  • flex-wrap: wrap
  • gap: 20px

↕️ Vertical Stack

  • display: flex
  • flex-direction: column
  • gap: 15px

🔄 Equal Columns

  • flex: 1 on each item
  • or flex-basis: 0

🎨 Navbar

  • display: flex
  • justify-content: space-between
  • align-items: center

📊 Main Axis vs Cross Axis

When flex-direction is "row":

  • Main Axis = Horizontal (left to right)
  • Cross Axis = Vertical (top to bottom)
  • justify-content = Horizontal alignment
  • align-items = Vertical alignment

When flex-direction is "column":

  • Main Axis = Vertical (top to bottom)
  • Cross Axis = Horizontal (left to right)
  • justify-content = Vertical alignment
  • align-items = Horizontal alignment
