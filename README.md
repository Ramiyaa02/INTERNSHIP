🌐 Responsive Portfolio Website – Internship Project
📌 Project Overview

This project is a responsive portfolio website created as part of the HTML/CSS Developer Trainee Program.
The website showcases a basic portfolio layout including sections such as Home, About, Projects, and Contact.

The goal of this project was to apply the concepts learned during the internship such as HTML structure, CSS styling, responsive design, Flexbox, Grid, and website deployment.
