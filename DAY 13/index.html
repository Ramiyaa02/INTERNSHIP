Learn how to create responsive designs that adapt to any screen size

What are Media Queries?

Media Queries are a powerful CSS feature that allows you to apply styles based on device characteristics like screen width, height, orientation, and resolution. They are the foundation of responsive web design.

🔑 Key Concept: @media Rule

@media (condition) {
  /* CSS rules */
}

1. Basic Media Query Syntax

max-width: Small screens (Mobile)

Resize your browser to see the color change below 768px

I change color when screen is 768px or less!

min-width: Large screens (Desktop)

Resize your browser to see the color change above 1024px

I change color when screen is 1024px or wider!

2. Common Breakpoints

Breakpoints are the widths where the layout changes. Here are common device sizes:

📱 Mobile

max-width: 600px

📲 Tablet

max-width: 768px

💻 Laptop

min-width: 1024px

🖥️ Desktop

min-width: 1200px

📋 Quick Reference Summary

Media Query Types

  • @media (max-width: x) - Small screens
  • @media (min-width: x) - Large screens
  • @media (orientation: portrait)
  • @media (orientation: landscape)

Common Breakpoints

  • Mobile: 600px
  • Tablet: 768px
  • Laptop: 1024px
  • Desktop: 1200px

Best Practices

  • Use mobile-first (recommended)
  • Use em/rem for breakpoints
  • Test on real devices
  • Keep it simple!
