📋 CSS Grid Quick Reference Guide

Your complete cheat sheet for CSS Grid Layout

🔧 Container Properties

display: grid

Enables grid layout for the container. All direct children become grid items.

/* Makes the element a grid container */
.container {
  display: grid;
}

grid-template-columns

Defines the number and size of columns in the grid.

px % fr auto repeat()
/* Three equal columns */
.container {
  grid-template-columns: 1fr 1fr 1fr;
}

/* Using repeat() */
.container {
  grid-template-columns: repeat(3, 1fr);
}

grid-template-rows

Defines the number and size of rows in the grid.

/* Fixed first row, auto second */
.container {
  grid-template-rows: 100px auto;
}

grid-template-areas

Creates named areas for easy layout structuring.

/* Classic website layout */
.container {
  grid-template-areas:
    "header header"
    "main sidebar"
    "footer footer";
}

/* Assign to item */
.header { grid-area: header; }

grid-gap / gap

Sets the space between grid rows and columns.

/* 20px gap for both */
.container { gap: 20px; }

/* Separate gaps */
.container {
  row-gap: 10px;
  column-gap: 20px;
}

justify-content

Aligns grid columns within the container (horizontal).

start end center space-between space-around space-evenly

align-content

Aligns grid rows within the container (vertical).

start end center space-between space-around stretch

📦 Grid Item Properties

grid-column

Defines which columns an item spans.

/* Span 2 columns */
.item { grid-column: 1 / 3; }

/* Span all columns */
.item { grid-column: 1 / -1; }

grid-row

Defines which rows an item spans.

/* Span 2 rows */
.item { grid-row: 1 / 3; }

grid-area

Assigns item to a named grid area OR specifies position.

/* Using named area */
.item { grid-area: header; }

/* Shortcut: row-start / col-start / row-end / col-end */
.item { grid-area: 1 / 1 / 3 / 3; }

justify-self

Horizontal alignment for a single item.

start end center stretch

align-self

Vertical alignment for a single item.

start end center stretch

🛠️ Special Functions

repeat()

Repeats a pattern of column/row sizes.

/* 4 equal columns */
grid-template-columns: repeat(4, 1fr);

minmax()

Defines a range between min and max size.

/* Minimum 100px, maximum 1 fraction */
grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));

auto-fit / auto-fill

Automatically creates columns based on available space.

/* Responsive columns without media queries! */
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

⚡ Quick Cheat Sheet

📊 Equal Columns

  • display: grid
  • grid-template-columns: repeat(3, 1fr)
  • gap: 20px

🌐 Responsive Grid

  • display: grid
  • grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))

📐 Page Layout

  • display: grid
  • grid-template-areas
  • "header header"
  • "main sidebar"
  • "footer footer"

🎯 Center Item

  • display: grid
  • place-items: center

📱 Mobile Layout

  • Single column on mobile
  • Use grid-template-areas
  • Switch to 1 column

🔄 Grid + Flex

  • Grid for page layout
  • Flex for component alignment
  • Best of both worlds!
